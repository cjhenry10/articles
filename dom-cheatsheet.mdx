# Introduction
The DOM, or Document Object Model, provides methods to add, change, and remove HTML elements using JavaScript.
#### Prerequisites
- HTML Elements, Variables, Strings, Objects

Each section will first show the code with the word placeholder, followed by examples of what can replace the placeholder. 
Next, there will be examples of how to use this method. This will usually involve using a variable, just remember that the variable names are your choice.

Here's a table of the methods detailed in this article.
|Goal | Method |
| --- | ----- |
| Select | `document.getElementById('your-element-id');{:js}` |
| Add a class | `yourHtmlElement.classList.add('your-class');{:js}` |
| Remove a class | `yourHtmlElement.classList.remove('your-class');{:js}` |
| Create | `document.createElement('div');{:js}` |
| Add to the page | `yourParentElement.appendChild(yourHtmlElement);{:js}` |
| Remove from the page | `yourParentElement.removeChild(yourHtmlElement);{:js}` |
| Adjust the inner HTML | `yourHtmlElement.innerHTML = 'Your new text';{:js}` |

# Select an HTML element
- `document.getElementById(placeholder);{:js}`
- Replace `placeholder{:js}` with a string that matches the id of the element you want to select, such as `'about'{:js}` or `'menu-item'{:js}`.

### Examples
The code below is part of the HTML file.
```html title="index.html" {3} showLineNumbers{4}
<body>
  <h1>Hello</h1>
  <div id="about-section"></div>
</body>
```
The code below selects the `<div>{:html}` element on line 6 using its id `'about-section'{:js}` and stores it in a variable named `about{:js}`.
```js title="script.js"
const about = document.getElementById('about-section');
```

# Add a Class to an HTML Element
- `placeholder1.classList.add(placeholder2){:js}`
- Replace `placeholder1{:js}` with your HTML element.
- Replace `placeholder2{:js}` with the class you want to add as a string, such as `'container'{:js}` or `'serif-header'{:js}`.
### Examples
The code below is in the HTML file.
```html title="index.html" {2}
<body>
  <h1 id="greeting">Hello</h1>
  <div id="home"></div>
</body>
```
To add the `'serif-header'{:js}` class to the `<h1>{:html}` on line 2, use the code below.
```js title="script.js"
// select the h1 by it's id
const h1Element = document.getElementById('greeting');
// add a class to the h1
h1Element.classList.add('serif-header')
```
Alternatively, you can do this without a variable.
```js title="script.js"
//    select the element             add a class
document.getElementById('greeting').classList.add('serif-header');
```
After either of the above JavaScript code examples run, the HTML in the browser will be updated. This will not change the HTML code you wrote in your code editor.
```html title="index.html" {2}
<body>
  <h1 id="greeting" class="serif-header">Hello</h1>
  <div id="home"></div>
</body>
```

# Create an HTML Element
- `document.createElement(placeholder);{:js}`
- replace `placeholder{:js}` with an HTML tag as a string, such as `'div'{:js}`, `'img'{:js}`, or `'section'{:js}`

### Examples
The code below creates a new `<div>{:html}` element and stores it in a variable named `newElement{:js}`. The element is not on the page yet.
```js title="script.js"
const newElement = document.createElement('div');
```

The code below creates a new `<p>{:html}` element and stores it in a variable named `newParagraph{:js}`. The element is not on the page yet.
```js title="script.js"
const newParagraph = document.createElement('p');
```
---
# Add an Element to the Page
- `placeholder1.appendChild(placeholder2){:js}`
- Replace `placeholder1{:js}` with your parent element, or the element that you want your element to be inside of.
- Replace `placeholder2{:js}` with your child element, or the element that will go inside the other element.

### Examples
The code below is part of the HTML file.
```html title="index.html" {1-3} showLineNumbers{20}
<div id="about-section">

</div>
<div id="contact-section">

</div>
```
To add a new `<p>{:html}` element to the about section, use the code below.
```js title="script.js"
// create the new element
const newParagraph = document.createElement('p');
// select the parent element
const aboutSection = document.getElementById('about-section');
// add the new element to the parent element
aboutSection.appendChild(newParagraph);
```

After the code above is executed, the HTML in the browser will be updated. This will not change the HTML code written in your code editor.
```html title="index.html" {2} showLineNumbers{20}
<div id="about-section">
  <p></p>
</div>
<div id="contact-section">

</div>
```

# Remove an Element from the Page
- `placeholder1.removeChild(placeholder2){:js}`
- Replace `placeholder1{:js}` with your parent element, or the element that surrounds the element you wish to remove.
- Replace `placeholder2{:js}` with your child element, or the element that you want to remove from the page.
### Examples
The code below is part of the HTML file.
```html title="index.html" {2} showLineNumbers{20}
<div id="about-section">
  <p id="about-paragraph">This is a paragraph in the about section.</p>
</div>
<div id="contact-section">

</div>
```
To remove the `<p>{:html}` element on line 21, use the code below.
```js title="script.js"
// select the element to be removed
const paragraph = document.getElementById('about-paragraph');
// select the parent element
const aboutSection = document.getElementById('about-section');
// remove the paragraph from the about section
aboutSection.removeChild(paragraph);
```
After the code above is executed, the HTML in the browser will be updated. This will not change the HTML code written in your code editor.
```html title="index.html" {2} showLineNumbers{20}
<div id="about-section">

</div>
<div id="contact-section">

</div>
```

# Adjust the innerHTML of an Element
- Set the inner HTML:
  - `placeholder1.innerHTML = placeholder2;{:js}`
  - Replace `placeholder1{:js}` with the element whose `innerHTML{:js}` you want to adjust.
  - Replace `placeholder2{:js}` with your desired HTML code as a string.
- Get the inner HTML:
  - `let placeholder3 = placeholder4.innerHTML{:js}`
  - Replace `placeholder3{:js}` with a variable name that will store the inner HTML.
  - Replace `placeholder4{:js}` with the HTML element whose inner HTML you want to access.
### Examples
The code below is part of the HTML file.
```html title="index.html" {4-6} showLineNumbers{20}
<div id="about-section">
  <p id="about-paragraph">This is a paragraph in the about section.</p>
</div>
<div id="contact-section">

</div>
```
To edit the `innerHTML{:js}` of the contact section, use the code below.
```js title="script.js"
document.getElementById('contact-section').innerHTML = '<h1>Contact</h1>';
```
After the code above is executed, the HTML in the browser will be updated. This will not change the HTML code written in your code editor.
```html title="index.html" {5} showLineNumbers{20}
<div id="about-section">
  <p id="about-paragraph">This is a paragraph in the about section.</p>
</div>
<div id="contact-section">
  <h1>Contact</h1>
</div>
```
To get the `innerHTML{:js}` of an element, use the code below.
```js title="script.js"
let pText = document.getElementById('about-paragraph').innerHTML;
console.log(pText);
```
The variable `pText{:js}` in the code above is now storing the `innerHTML{:js}` of the paragraph in the about section. The output to the console would read: "This is a paragraph in the about section."
